=========OrderProcessing=========

Unvalidated Order -> UnvalidatedProductList
Invalidated Order -> UnvalidatedProductList, reason
Validated Order   -> ValidatedProductList
Available Order   -> AvailableProductList

Events:

OrderProcessingFailedEvent   -> reason
OrderProcessingSuccessEvent  -> AvailableProductList


=========Billing=========

UnpayedOrder       -> AvailableProductList
PayByCardOrder     -> AvailableProductList, total
PayByCashOrder	   -> AvailableProductList, total
PaymentFailedOrder -> reason
PayedOrder	   -> AvailableProductList, total

Events:

BillingFailedEvent  -> reason
BillingSuccessEvent -> AvailableProductList, total


=========Shipping=========

UnshippedOrder -> AvailableProductList, total
ShippedOrder   -> reason

Events:

ShippingSuccessEvent -> reason
